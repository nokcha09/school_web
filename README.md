# 개요

## 이 깃은 무었인가?

이 깃은 선린인터넷 고등학교 116기 콘텐츠 디자인과의 수업내용을 기록해둔 곳 입니다.</br>
공식적인 기록이 아닌 제가 정리하기 위해 올려둔 것입니다.

### 수업내용을 깃에 올리는 이유

수업을 배움에 있어 전에 있던 내용을 기억하고 싶을 때가 있으나 항상 파일이 없어 찾아 볼 수가 없었습니다. </br>
깃에 정리함에 있어서 나중에 복습하고 공부하기 위해 올리게 되었습니다.

## 이 문서를 보는 방법

### 일자별 정리 내용

이 README 문서에 아래를 내려 보면 일자별 정리 텝이 있습니다. </br>
해당 내용에 대한 더 자세한 설명을 보고 싶으면 상세설명을 누르게 된다면,</br>
상세한 설명과 내용이 적혀 있습니다.

### 상세 설명 작성 시점

학교 수업을 들으면 정리하기 때문에 상세 설명을 바로 바로 기제 할 수 없는 문제가 있습니다. </br>
그래서 해당 설명은 수업을 듣고 저의 방식대로 정리해서 설명한걸로 특정 내용이 누락되거나 빠질 수 있습니다.

### 저작권

본 내용에 있는 모든 이미지의 경우 구글에서 이미지 다운을 통해 가져온 사진으로,</br>
저작권에 문제가 생길 수 있는 이미지를 사용 하였습니다. </br>
수업 중 이미지를 빠르게 구해야 하기 때문에 무분별한 저작권 사용하게 되었습니다. </br>
사과의 말씀 드리며, 시간날때 마다 이미지를 제가 직접 찍은 사진으로 변경할 예정입니다.

</br></br>
